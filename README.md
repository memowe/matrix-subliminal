# matrix-subliminal

Matrix screen with subliminal messages

## Run

```bash
$ cabal run matrix-subliminal -- "The end is near!" "YOLO"
```

## Install

```bash
$ cabal install
```

```bash
$ matrix-subliminal "The end is near!" "YOLO"
```
